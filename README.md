## Proyecto desarrollado en 2023 junto a 4 compañeros. Juego creado en C# con control de versiones en GitHub (integración continua). Incluye una API en Java con una base de datos MySQL.

Arc, el personaje es un joven aprendiz de magia en busca de su poder mágico para completar su formación.
Con un fuerte deseo de aprender y una pasión por la magia, está decidido a superar cualquier obstáculo que se le presente en su camino para alcanzar su destino y convertirse en un mago poderoso.
